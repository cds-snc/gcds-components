# Map<br/>`<mapml-viewer>`

The `<mapml-viewer>` component enables slippy maps with one or more `<map-layer>` children.

The `<map-caption>` element can be used to provide a screen reader friendly caption or title
for the map.

You can "hide" a layer, such as a basemap layer (in the layer control), while ensuring that is always visible
on the map. This can be useful to reduce the cognitive load of your map application,
and may improve the map's accessibility for some users.
