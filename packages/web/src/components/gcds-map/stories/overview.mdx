# Map<br/>`<mapml-viewer>`

The `<mapml-viewer>` component enables slippy maps with one or more `<map-layer>` children.

The `<map-caption>` element can be used to provide a screen reader friendly caption or title
for the map.

### A map with a "hidden" basemap and a single thematic layer exposed in the layer control

You can "hide" a basemap layer from the layer control, while ensuring that it is always visible
on the map. This can be useful to reduce the cognitive load of your map application,
and may improve the map's accessibility for some users.

<Canvas of={Map.HiddenBasemap} story={{ inline: true }} />

### A map including default styled GeoJSON features

The MapML viewer library has built-in support for converting GeoJSON into MapML
features and layers.  You can use custom JavaScript functions to provide styles
for your features.
